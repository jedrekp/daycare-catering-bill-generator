Welcome to Daycare Catering Bill Generator

As a headmaster you are given full access to all sections of this application.

Visit the child profiles facemodule in order to create new child profiles and display/edit the existing ones. You may also assign new catering options to a child and adjust their monthly attendance by entering that child's profile page.

Daycare groups groupsection is a place where you can manage existing daycare groups and add new ones. Go to a single group page to edit it's properties and assign new children or group supervisor to that group.

Attendance how_to_regsection is used to track daily attendance for each daycare group. Select date and daycare group, then mark attendance for all it's members. If you need to adjust monthly attendance for a signle child, it might be easier to do it by visiting their profile page instead.

You may use the Group supervisors supervisor_accountmodule to create and delete group supervisor user accounts. By visiting this section, you can also assign group supervisor to a daycare group or revoke their group assignment. Note that group supervisors are only able to track attendance for children that are assigned to the same daycare group.

Go to Catering options restaurantmodule to add new catering options and manage the price and name of existing ones. You can also disable and enable individual catering options in this section. Please note, that while disabled catering options can no longer be assigned, they will not be removed from children to whom they are assigned already.

Catering bills receiptsection is used for generating catering invoices and sending them to parents via email. Each catering bill will be generated automatically based on child's attendance in given month, as well as catering options that are in effect for that period. Additionally this module allows you to dsiplay already generated catering bills and generate corrections when necessary.

As a group supervisor you are permited to track attendance of children assigned to your daycare group. Additionally you have a read-only access to child profiles and daycare groups sections.

Use the attendance how_to_regsection to track daily attendance for your daycare group. Select the date and the group that is being supervised by you, then mark attendance for all it's members.

If you need to adjust attendance for a signle child, you can do so by visiting the child profiles facemodule and entering that child's profile page. This allows you to quickly modify that child's attendance for entire month. Note that this will only work if that child is assigned to your daycare group.

You may also access a child profile page by going to the daycare groups group section and selecting the daycare group to display the list of all its members.
